
🌾 究極のC#農業シミュレータ (AgriSimComplete)
C# Windows Forms で構築された、本格的な農業経営シミュレーションゲームです。
限られた資金と土地を活用し、100日間で最高の純利益を目指します。

🌟 主な特徴
リアルタイム・農場マネジメント:

土地の拡張、種まき、肥料散布、病害虫駆除など、本格的な農作業プロセスを再現。

カイワレからトマトまで、成長速度と利益率の異なる作物を戦略的に選択。

ダイナミック・シーズンシステム:

春・夏・秋・冬の4シーズン制。季節ごとに背景色が変化し、作物の市場価格が激しく変動します。

（例：冬のトマトは高級品として通常の2.5倍で売却可能！）

病害とパンデミック:

放置すると隣接する畑に病気が広がる「感染拡大アルゴリズム」を搭載。農薬による迅速な対処が求められます。

リッチなゲーム演出:

収穫時のボタンシェイク演出、利益のポップアップ表示、BGM/SE対応など、心地よい操作感を実現。

セーブ＆ロード機能:

System.Text.Json を使用したシリアル化により、経営状態を瞬時に保存。

🛠 技術的なポイント
堅牢な収穫ロジック: 非同期的なイベントによる多重クリック（二重収穫）を防ぐため、データを一時変数に退避させてから処理する安全な設計を採用。

オブジェクト指向設計: Crop（作物）、Plot（耕作地）、SaveData（保存形式）といった明確なクラス構造。

UI/UX最適化: DoubleBuffered による描画のちらつき防止や、FlowLayoutPanel による動的な土地拡張への対応。

🚀 クイックスタート
環境準備: .NET Framework または .NET 6.0+ が動作する環境。

ビルド: Visual Studio で AgriSimComplete.sln を開き、ビルドを実行。

アセットの配置: 実行ファイル（EXE）と同じフォルダに以下の音声ファイルを配置してください（任意）。

se_harvest.wav (収穫音)

bgm_farm.wav (環境音)

🕹 ゲームルール
期間: 100日間

勝利条件: 最終スコア（累計獲得金額）の最大化。

敗北条件: 資金が -5,000円 を下回ると倒産（ゲームオーバー）。

ヒント: 10日ごとの「納税日」に備えて現金を確保しておくことが重要です。

📂 クラス構成
Form1.cs: メインのゲームループ、UI制御、演出ロジック。

SaveData.cs: 永続化のためのデータモデル。
